====================================================================
TROUBLESHOOTING
====================================================================

Backend not starting:
→ Check Railway logs
→ Verify DATABASE_URL format
→ Ensure gunicorn installed (in requirements.txt)

Frontend can't reach backend:
→ Check CORS_ORIGINS matches Vercel URL
→ Check NEXT_PUBLIC_API_URL is set
→ Open browser console for errors

Database connection failed:
→ Verify DATABASE_URL from Railway PostgreSQL service
→ Check pgvector extension (Railway auto-installs)

Neo4j connection failed:
→ Verify URI format: neo4j+s:// (not bolt://)
→ Check firewall allows connection
→ Test in Neo4j Browser first
