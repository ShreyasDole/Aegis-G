# Methodology / Working

## Step-by-step working of system
1. Data ingested from API or UI.
2. Adversarial shield denoises the text.
3. ONNX local inferencer runs model prediction.
4. SHAP logic yields a token heatmap.
5. All results synthesized into a final report.

## Algorithms used
- Categorical Cross-Entropy for multi-class classification.
- Approximate Nearest Neighbor (ANN) via Cosine Similarity for RAG.
